# Reorganisation plan for cogcache tree

## Archive organic development docs

Move the original design docs (created during exploratory sessions) to an archive.
These have historical value but are superseded by the consolidated documents.

```bash
# From cogcache root:
mkdir -p inst/doc-archive

# Archive original exploration docs
mv inst/design-docs/cog-duckdb-prototype-sketch.md  inst/doc-archive/
mv inst/design-docs/cog-duckdb-cache.qmd             inst/doc-archive/
mv inst/design-docs/gdal-warper-audit.md              inst/doc-archive/
mv inst/design-docs/rust-warper-design.md             inst/doc-archive/
mv inst/design-docs/warp-mapping-prototype.qmd        inst/doc-archive/

# Archive the earlier checkpoint too
mv inst/design-docs2-checkpoint/GDAL_WARP_ARCHITECTURE.md \
   inst/doc-archive/GDAL_WARP_ARCHITECTURE_checkpoint2.md

# Clean up empty dirs
rmdir inst/design-docs inst/design-docs2-checkpoint
```

## Install new consolidated docs

```bash
mkdir -p inst/design-docs

cp VWARP_VISION.md           inst/design-docs/
```

## Exports (R functions)

Current exports from `extendr-wrappers.R`:

| Function | Purpose | Module |
|----------|---------|--------|
| `rust_decode_tile` | DEFLATE+predictor decode | lib.rs |
| `rust_fetch_decode_tile` | HTTP range + decode | lib.rs |
| `rust_gen_img_proj_transform` | exact coordinate transform | transform.rs |
| `rust_warp_scanline` | exact per-pixel warp | warp.rs |
| `rust_warp_approx` | approx warp (production path) | approx.rs + warp.rs |
| `rust_compute_source_window` | source footprint planning | source_window.rs |
| `rust_warp_map` | legacy: index-based mapping | lib.rs |
| `rust_apply_warp` | legacy: apply index mapping | lib.rs |

The `rust_warp_map` and `rust_apply_warp` functions are legacy — they
predate the scanline warp architecture. Keep for now, deprecate later.
